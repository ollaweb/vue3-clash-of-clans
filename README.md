### Description

Study project. Vue-router setup. Redirect to Not Found page.
Project made by learning Vue 3 [in this course.](https://tocode.ru/courses/vuejs-3-s-nulya-do-rezultata/)

#### In this project:

- using options API;
- using vue-router;
- dynamic redrawing without page reloading;
- reusing components;
